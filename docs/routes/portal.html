Documentação da API do Portal

Endpoints

1. Listar Cadernos do Portal

Endpoint: /portal

Método: GET

Autenticação: Nenhuma

Descrição: Lista todos os cadernos do portal que não estão com status "Pendente".

Resposta de Sucesso:

200 OK: Lista de cadernos no portal.

Resposta de Erro:

400 Bad Request: { "message": "Ocorreu um erro ao processar sua solicitação" }

2. Detalhes de um Caderno do Portal

Endpoint: /portal/book/:idBook

Método: GET

Autenticação: Nenhuma

Descrição: Obtém os detalhes de um caderno específico do portal.

Parâmetros:

  • idBook (integer, obrigatório): ID do caderno.

Resposta de Sucesso:

200 OK: Detalhes do caderno, incluindo estática, features e resumo dos testes.

Resposta de Erro:

404 Not Found: { "message": "Caderno não encontrado" }

400 Bad Request: { "message": error.message }

3. Listar Cadernos Recentes

Endpoint: /recentBooks

Método: GET

Autenticação: Nenhuma

Descrição: Lista os cadernos recentes aprovados, limitados pelo parâmetro limit.

Parâmetros:

  • limit (integer, opcional): Número máximo de cadernos a serem retornados. Se não for fornecido ou for inválido, o valor padrão é 5.

Resposta de Sucesso:

200 OK: Lista de cadernos recentes aprovados.

Resposta de Erro:

404 Not Found: { "message": "Caderno não encontrado" }

400 Bad Request: { "message": "Ocorreu um erro ao processar sua solicitação" }
